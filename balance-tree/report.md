# BalanceTree

#### 介绍
1.  利用课本提供代码, 实现 AVL Tree 和 Splay Tree.
2.  分别使用 AVL Tree 和 Splay Tree 实现 BST 排序算法的实现代码

#### 软件架构
1.  Makefile编译规则
    直接运行
    make
    产生可执行文件
    test
    用于测试程序;
2.  main.cpp主流程
    编译后产生
    test;
3.  BalanceTree.h头文件
    直接使用课本提供的代码；
4.  report.md
    解释项目设计思路和测试说明;
5.  run脚本文件
    用于测试
	运行
    bash run
    对排序结果进行验证.
